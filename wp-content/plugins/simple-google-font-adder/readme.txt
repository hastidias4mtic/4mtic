== Description ==

Add the Google font link of your choice and it will be automatically added in your text editor.
Tested with WPBakery, Gutenberg, Advanced TinyMCE Editor

= Features include: =

* Easy to use: no need to add a file by hand (classic method add each .ttf + add the font family in the css)
* Adds the font to the font selection without adding an addon to the text editor.
* Works for sur with: wpbakery, visual composer, gutenberg (tested)
* Selective loading of fonts.
(Adds the precise font without adding the +990 other fonts so it\'s easy to find the font you want to add )
* Real-time font preview added
* SSL and HTTPS compatible.
* Tested with PHP7.* and latest version of wordpress.
* Works with any WordPress theme. No coding required.
* Ready for translation.
* Free, open source

== Installation ==
1. Install Simple Google Font Adder either via the WordPress.org plugin directory, or by uploading the files to your server
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Go to the Add Google Font settings and set your fonts.
Full tutorial here : https://github.com/f-paul/google-font-adder
